# Amazon-Sales-Dashboard (Microsoft Excel)

### An interactive Sales Dashboard built in Microsoft Excel to analyze Amazon sales performance. The dashboard transforms raw sales data into meaningful business insights using interactive filters, KPIs, and visual charts.

### Project Overview

This dashboard helps track overall business performance by providing a clear view of sales, profit, product performance, category analysis, sales type, payment mode, and monthly/daily sales trends. Users can filter the report by Year, Month, Sales Type, and Payment Mode for dynamic analysis.

### Key Features

KPI Cards (Total Sales, Total Profit, Profit %)

Interactive Slicers (Year, Month, Sales Type, Payment Mode)

Monthly Sales Trend Analysis

Category-wise Performance (Treemap)

Top Product & Top Category Highlights
